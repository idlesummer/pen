# Pen

A Next.js-inspired framework for building Ink (React terminal UI) applications.

## Installation

\`\`\`bash
npm install github:yourusername/pen
\`\`\`

## Usage

In your project with a \`src/app/\` directory:

\`\`\`bash
# Development mode
npx pen dev

# Production mode
npx pen start
\`\`\`

## Development

\`\`\`bash
# Install dependencies
npm install

# Build
npm run build

# Link for local testing
npm link
\`\`\`

## Testing Locally

See the \`pen-test\` repository for a minimal test application.
